# Genome_TE_annotation_pipeline
Pipeline for parsing FASTA and GFF files to extract strand-specific nucleotide composition.

# Genome Annotation Pipeline

## Objective
This pipeline parses genomic FASTA (.fna) and annotation (.gff) files to extract strand-specific nucleotide composition.

## Features
- FASTA parsing
- GFF annotation parsing
- Strand (+/-) gene extraction
- Nucleotide frequency calculation
- CSV output generation
- Visualization of strand composition

## Technologies
- Python
- Biopython
- pandas
- matplotlib

## Input
- genome.fna
- genome.gff

## Output
- strand_composition.csv
- strand_composition_plot.png
